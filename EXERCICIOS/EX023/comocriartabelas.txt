HIERARQUIA DE TABELAS (simples)
    TABLE => tabela
      TABLE ROWS = linha de tabela
        TABLE HEADER = cabeçalho da tabela
        TABLE DATA = dado da tabela

pelo w3c não e obrigatorio do fechamento das tags <td>.

se a HIERARQUIA não for obedecida a liha irá aparecer antes ou depois da tabela.

border-collapse: collapse;(dados colados, colapsados um ao outro.)

border-collapse: separete; (dados separados.)

vertical-align: middle; para alinhar verticalmente o dado da tabela ao centro da tabela ou no meio da tabela.

<th> titulo <th>
<td> um dado <td>

no <th> se o dado a baixo da linha estiver em coluna uso scope="col", se o dado estiver na mesma linha que o titulo uso scope="row"

para que faça sentido para o dispositivos de busca.

tbody > tr:nth-child para dizer que na minha tabela no corpo dela de duas em duas ira ter um estilo ou cor diferente.

para o cabeçalho da tabela seguir o resto da tabela quando a tabela e muito grande usamos no style 
table{
  position:relative
}
 thead > tr > th{
            position: sticky;
            top: -1px;
            background-color:gray;
        }

para que uma linha se preencha ou mescle no lugar da outra no caso ocupe espaço da outra usa por exemplo
<td rowspan="2">
para que ela se preencha em uma coluna usa-se 
<td colspan="2".

nos th's pode usar a função scope que diz o que o titulo está falando 
para somente uma coluna usamos scope="col"
para somente uma linha usamos scope="row"
para quando é mais de uma coluna usamos scope="colgroup"
para quando é mais de uma linha usamos scope"rowgroup".

Para que eu mude a configuração de uma coluna toda posso usar a tags ex.

<colgroup>
    <col class="cnome"
    <col class="cidade"
</colgroup>

caso queira que a mesma configuração vá para varias colunas uso a tags

<colgroup>
    <col class="cgrupo span="3" (aqui no caso para configurar as 3 colunas)
</colgroup>

posso tambem criar uma classe como:

.amarelo{
  background-color: yellow
}
.vermelho{
  background-color: red
}

e colocar direto em colgroup diretamente a classe na coluna:

<colgroup>
    <col class="amarelo">
    <col class="vermelho">
    <col class="amarelo">
    <col class="vermelho">
</colgroup>

Para que meu site fique responsivo e minha tabela quando diminuo o tamanho da tela ela crie um ponto de rolagem so para ela não deformando meu site eu envolvo toda minha tabela em uma div. 
e crio a seguinte tags.

<div id="container">
  <table>
  </table>
</div>

crio em style uma condição de rolagem para essa div.

div#container{
            overflow-x: auto;
        }
 
 aqui estou dizendo que essa tabela vai ter uma rolagem em x ou seja para o lado.
