o iframe e como se conseguisse abrir uma uma tela ou uma guia dentro do meu site
basta colocar a tag <iframe src"o site que vc quiser" frameborder="0"><iframe>.
entre essa tag por medida de segurança caso o dispositivo usado não abre o iframe. usa-se a tag <a> criando um link

ex:

<iframe src="https://gustavoguanabara.github.io" frameborder="0">
    
    <a href="https://gustavoguanabara.github.io" target="_blank"> no github</a>
    
</iframe>

tamanho padrão de iframe e de 300x150px

esse tamanho posso alterar, para alterar altura uso dentro de iframe height:

<iframe src="https://www.cursoemvideo.com" frameborder="0" height="500"></iframe>

para alterar largura uso widht:

<iframe src="https://gustavoguanabara.github.io" frameborder="0" width="500"></iframe>

para alterar os dois:

<iframe src="https://gustavoguanabara.github.io" frameborder="0" height="500" width="500"></iframe>

as css são soberanas as configurações que estão nelas sempre ira prevalecer.

ex:

iframe{
        width: 400px;
        height: 400px;

        }

em alguns navegadores se eu quiser que minha pagina no iframe não tenha rolagem uso a tag scolling="no" dentro da tag <iframe></iframe> ex:

<iframe src="https://www.cursoemvideo.com" frameborder="0" height="500" scolling="no"></iframe>

para que eu faço um link abrir em um frame na minha propria pagina eu nomeio meu iframe, e depois em target no meu link coloco o nome que coloquei no meu frame. ex:

<li><a href="paginas-extras/pag001.html" target="frame">Primeira página</a></li>


<iframe id="tela" name="frame">
        <p>Infelizmente seu navegador não é compatível com isso.</p>

    </iframe>

no iframe posso criar um codigo dentro dele se eu quiser usando a tag srcdoc="" entre essas aspas crio o meu codigo.
ex:
<iframe id="tela" name="frame" srcdoc="<h1>Escolha uma das opções acima</h1><p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Qui odio ut ipsum quo minus, tenetur facere praesentium id, ab, molestiae eveniet maiores excepturi? Cumque quod fugiat necessitatibus, ex doloribus est? Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>"></iframe>.

para uma proteção melhor para o seu site, uso a tag sandbox="sandbox" no iframe ex:

 <iframe src="paginas-extras/pag004.html" sandbox="sandbox" referrerpolicy="no-referrer"></iframe>

 sandbox= allow quer dizer permitir dae se quiser permitir algo coloca allow e o que quiser permitir na frente da tag exemplo se quiser permitir script faço:
 
 sandox="allow-script"
